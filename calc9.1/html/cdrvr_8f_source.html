323 C PROGRAM STRUCTURE
324 C
325 C Perform the geometry and time calculations.
326 C The basic coordinate system is referenced to the Epoch of 2000.0 and is a
327 C right-handed Cartesian system oriented to the mean celestial pole and mean
328 C equator of that epoch. The nominal origin is the solar system barycenter.
329 C There is also an Earth fixed coordinate system which is a right-handed
330 C Cartesian system oriented to the mean geographic pole of 1900-1906 and the
331 C Greenwich Meridian. The nominal origin is the Earth's center of mass. The
332 C basic unit of time is the coordinate second as used by the PEP Tape. UTC,
333 C AT, AND UT1 are also used. The geometry of the observation is calculated
334 C with an accuracy goal of 0.1 picoseconds of delay. In doing the
335 C calculations for the geometry, much of the work neccesary for the
336 C computation of model contributions to delay and delay rate and partials of
337 C delay and delay rate with respect to model parameters is also done.
338 C Matrices which represent coordinate rotations (precession, nutation,
339 C diurnal spin, diurnal polar motion, and wobble) and their CT time
340 C derivatives are stored as (3,3,N) arrays, where N indixes the N-1'th time
341 C derivative. The subroutines suffixed G are sections of model modules. The
342 C other subroutines may be considered utilities and either superseed or
343 C incorporate many present PEP routines.
344 C
345 C Call SITG for the geographical site data. SITG provides the following
346 C geocentric information for each observing site: the antenna axis offsets
347 C (AXOFF), the antenna types (KAXIS), the crust fixed site vectors (CFSITE),
348 C the crust fixed baseline vector (CFBASE), the crust fixed site normal unit
349 C vectors (CFSITN), the geodetic latitudes (SITLAT), the site east
350 C longitudes (SITLON), the spherical Earth radii, the partial derivatives of
351 C the crust fixed site vector components with respect to the geodetic
352 C latitudes (CFLAT) and east longitudes (CFLON), the rotation matrices which
353 C rotate the topocentric site reference system to the geocentric system at
354 C each site (TCTOCF), and the zenith tropospheric path delays at each
355 C observing site. SITG is the only routine which 'knows' which two sites are
356 C involved in the observation. All other routines merely work with site #1
357 C and site #2.
358 C
359  CALL sitg (axoff, cfbase, cflat, cflon, cfsite, cfsitn, kaxis,
360  * oceamp, ocephs, sitlat, sitlon, sitrad, tctocf, rttocf,
361  * zpath, sitheight, geolat)
362 C
363 C Call UTCTM for the UTC time fraction of the UTC day (UTC) and for
364 C the Julian Date at zero hours UTC of the date in question (XJD).
365  CALL utctm (utc, xjd)
366 C
367 C Call STRG for the J2000.0 unit vector in the direction of the
368 C radio source. (STAR)
369  CALL strg (xjd, utc, star)
370 C
371 C Call ATIME for the atomic time fraction of the atomic time day (AT) and
372 C for the partial derivative of the UTC time with respect to the atomic
373 C time (DUTCAT).
374  CALL atime (utc, xjd, at, dutcat)
375 C
376 C Call CTIMG for the coordinate time fraction of the coordinate time day at
377 C site #1 (CT), the partial derivative of the atomic time with respect to
378 C the coordinate time (DATDCT), and the partial derivative of the long
379 C period terms in the 'AT minus CT' offset with respect to the coordinate
380 C time (DLPGR).
381  CALL ctimg (at, cfsite, sitlon, utc, xjd, ct, datdct, dlpgr)
382 C
383 C Compute epoch and compare with previous observation. If same, set
384 C TSKIP=1, otherwise TSKIP=0. If TSKIP=1, then we can skip many steps in
385 C the geometry subroutines.
386  tjd = xjd + ct
387  obsdif = dabs(tjd - sjd)
388  IF (obsdif .lt. 2.d-10) THEN
389  tskip = 1
390  ELSE
391  tskip = 0
392  sjd = tjd
393  ENDIF
394 C
395 C Call PEP for the J2000.0 geocentric Sun (SUN) and Moon (XMOON) position
396 C and velocity vectors; the J2000.0 solar system barycentric Earth
397 C position, velocity, and acceleration vectors (EARTH); the other planets'
398 C (except Pluto) barycentric and geocentric positions and velocities.
399 C The solar system info comes from the DE/LE403 JPL Ephemeris by default.
400  CALL pep (xjd, ct, tskip, earth, sun, xmoon)
401 C
402 C Call NUTFA before NUTG and before UT1G to get epoch in centuries and
403 C the fundamental arguments for the nutation series.
404  IF (tskip .NE. 1)
405  * CALL nutfa (xjd, ct, cent, fa, fad)
406 C
407 C Call UT1G for the UT1 fraction of the UT1 day (UT1) and for the partial
408 C derivative of the UT1 time with respect to the atomic time (DUT1AT).
409  CALL ut1g (at, dutcat, utc, xjd, ct, fa, fad, cent, tskip,
410  * dut1at, ut1)
411 C
412 C Call NUTG for the nutation portion of the complete crust fixed to J2000.0
413 C rotation matrix and its CT time derivative (RN), the true obliquity of
414 C the ecliptic and its CT time derivative (EPS), and the mean obliquity at
