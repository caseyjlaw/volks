617 C

622  DO 600 n=1,3
623  DO 600 m=1,3
624  rn(m,n,2) = 0.0d0
625  600 CONTINUE
626  ENDIF

653  WRITE(6,8)' R2N ',r2n
654  WRITE(6,8)' R3N ',r3n

660  WRITE(6,8)' RN6 ',rn6
661  725 CONTINUE
662 C
663  WRITE ( 6, 9200 ) deps, dpsi, eps, epsmn, rn,
664  1 epsmnr
665  9200 FORMAT (1x,
666  1 "DEPS = ", 2 ( d30.16, 10x ), /, 1x,
667  2 "DPSI = ", 2 ( d30.16, 10x ), /, 1x,
668  3 "EPS = ", 2 ( d30.16, 10x ), /, 1x,
669  4 "EPSMN = ", 2 ( d30.16, 10x ), /, 1x,
670  5 "RN = ", 6 ( 3 ( d30.16, 10x ), /, 1x ),
671  9 "EPSMNR= ", d30.16, /, 1x)

918 C RESPECT TO PSI AND EPSILON AND PLACE IN DATA BASE.

1727  SUBROUTINE ls_nut( CENT, fa, fad, dpsi_ls, deps_ls )
1728  Implicit None
1729 C
1730 C Routine to compute the KSV_1996_3 luni-solar contributions the
1731 C nutations in longitude and obliquity. The KSV_1996_3 is based on:
1732 C
1733 C (1) The Souchay and Kinoshita Rigid Earth nutation series KSRE95. The
1734 C seven terms with duplicate arguments on the KS series have been
1735 C combined into single terms.
1736 C The arguments were:
1737 C l lp F D Om Period (days)
1738 C -1 0 0 1 0 411.78
1739 C 0 1 0 0 0 365.26
1740 C 0 -1 2 -2 2 365.22
1741 C 0 2 0 0 0 182.63
1742 C 0 0 2 -2 2 182.62
1743 C 0 -1 -2 2 -2 121.75
1744 C 0 0 0 1 0 29.53
1745 C The series has also been sorted in increasing order of amplitude of the
1746 C nutation. This is to minimize rounding error as the series is summed
1747 C in reverse order.
1748 C
1749 C (2) Estimates of the Retrograde FCN resonance factors from the Mathews
1750 C et al., nutation formulation (full complex estimates, and scaling
1751 C parameter R from the same theory. (The definition of complex values
1752 C here is Rr + iRi; in earlier versions the definition was Rr -iRi.
1753 C This change does not effect the nutation series coefficients). The
1754 C nutation amplitudes are still defined as ar - ai).
1755 C
1756 C The resonance factors used are:
1757 C Type Amplitude Frequency (cpsd)
1758 C Real Imag. Real Imag.
1759 C RFCN -.00011489752 .00000214130 -1.00231888314 -.00002920327
1760 C CW -.00057992000 .00000000000 .00253170000 .00000000000
1761 C R and R' 1.04901828112 -.00150732471 -.25517427386 .03965769073
1762 C
1763 C (3) The effects of annual modulation of geodetic precession.
1764 C The correction applied is
1765 C 0 1 0 0 0 -0.150 (correction to in-phase nutation in longitude).
1766 C
1767 C (4) A prograde annual nutation has been estimated along with the
1768 C resonance coefficients. This probably reflects the influence of
1769 C S1 atmospheric tide.
1770 C
1771 C (5) The free RFCN mode was estimated once every two years for the
1772 C data after 1984. (See values commented in eval_ls_nut. For the
1773 C last 6 years the values seem to be resonably stable.
1774 C
1775 C (6) The new Simons et al., fundamental arguments are used in this
