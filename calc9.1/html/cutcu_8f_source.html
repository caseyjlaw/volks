4 C 1. UTCTM

6 C 1.1 UTCTM PROGRAM SPECIFICATION
7 C
8 C 1.1.1 UTCTM is the utility which fetches the UTC time tag of the observation
9 C and computes the Julian date at 0 hours UTC of the date of the obs.
10 C
11 C 1.1.2 RESTRICTIONS - Computation of Julian date correct for 1901 to 2099 only.

15 C
16 C 1.2 UTCTM PROGRAM INTERFACE
17 C
18 C 1.2.1 CALLING SEQUENCE -
19 C INPUT VARIABLES: None
20 C OUTPUT VARIABLES:
21 C 1. UTC - THE UTC TIME AS A FRACTION OF THE UTC DAY. (SEC/SEC)
22 C 2. XJD - THE JULIAN DATE AT ZERO HOURS UTC OF THE DATE IN
23 C QUESTION. (DAYS)
24 C
25 C 1.2.2 COMMON BLOCKS USED -
26 C

98 C UTCTM Program Structure
99 C
100 C GET the time tag information from the database. Check for db errors.
101 C First try to get new time tag with 4-digit year. If not there, get old
102 C time tag with 2-digit year. Doesn't really matter though.
103  CALL geti ('UTC TAG4 ', itag, 5, 1, 1, ndo, kerr(1) )
104  If (kerr(1) .ne. 0)
105  * CALL geti ('UTC TAG ', itag, 5, 1, 1, ndo, kerr(1) )
106  CALL get4 ('SEC TAG ', tagsec, 1, 1, 1, ndo, kerr(2) )
107  DO 200 n = 1,2
108  nn = n
109  IF ( kerr(n) .EQ. 0 ) GO TO 200
110  CALL ckill (6hutctm , nn, kerr(nn) )
111  200 CONTINUE
112 C
113 C Compute the Julian date at 0 hours UTC for the year, month, day.
114 C Use function JDY2K to convert year, month, day to Julian date. Year can be
115 C either 2-digit or 4-digit.
116  iyy = itag(1)
117  im = itag(2)
118  id = itag(3)
119  xjd = jdy2k(iyy,im,id)
120 c write(6,'("UTCTM: ITAG(1), New XJD = ",i5,f12.2)') IYY,XJD
121 C
122 C Compute the UTC time as a fraction of the UTC day.
123  utc = ( dfloat( itag(4) ) * 3600.d0
124  1 + dfloat( itag(5) ) * 60.d0
125  2 + tagsec ) / secday
126 C
127 C Check for debug.
128  IF ( kutcd .EQ. 0 ) GO TO 500
129  WRITE ( 6, 9)
130  9 FORMAT (1x, "Debug output for subroutine UTCTM." )
131  WRITE(6,7)' ITAG ',itag
132  7 FORMAT(a,15i8/(9x,15i8))
133  WRITE(6,8) 'SECDAY',secday
134  8 FORMAT(a,4d25.16/(7x,5d25.16))
135  WRITE(6,8)' TAGSEC ',tagsec
136  WRITE ( 6, 9200 ) utc, xjd
137  9200 FORMAT (1x, "UTC = ", d30.16, /, 1x,
138  1 "XJD = ", d30.16 )
139 C
140  500 RETURN
141  END
