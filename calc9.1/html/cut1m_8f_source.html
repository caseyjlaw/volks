1693  Implicit none

1695 C Subroutine spline is the initialization section of a two subroutine
1696 C module used to do cubic spline interpolations. Spline is called each
1697 C time you change to a new set of tabular points. Subroutine splint4
1698 C is the second half and does the actual interpolation.
1699 C
1700 C Given arrays xa(i), and ya(i) for i=1 to n containing a tabulated
1701 C function ya(i) = f(xa(i)) with xa(1) < xa(2) <..< xa(n) and values
1702 C yp1 and ypn whichc are the first derivatives of the function f( ) at
1703 C points 1 and n, respectively, this routine returns an array
1704 C y2(i) for i=1 to n which contains the second derivatives of the
1705 C function f( ) at the tabulated points xa(i). If yp1 and/or ypn are
1706 C greater than or equal to 1.d30, the routine is signaled to set the
1707 C corresponding boundary condition for a natural spline, with zero
1708 C second derivatives on the boundary.
1709 C
1710 C The cubic spline has the following properties.
1711 C 1) It is continuous and exactly fits at the tabular values.
1712 C 2) The first and second derivatives are everywhere continuous,
1713 C even at the tabular points when points are added to and
1714 C deleted from the set of five points used to interpolate.
1715 C 3) The third derivative is not continuous, but rather a series
1716 C of disconnected constant values.
1717 C 4) The fourth derivative is zero.
1718 C 5) It requires a minimum of five points and values of the first
1719 C derivative at the two endpoints.
1720 C
1721 C In order to guarantee that this routine is only used in the modes in
1722 C which it has been tested, we require that the values of xa be exactly
1723 C one unit apart.

1731 C array. The xa's must be evenly spaced.
1732 C 2. ya(n) - Array of EOP values at the times corresponding to those
1733 C in the xa array. i.e. ya(i) = f(xa(i)).
1734 C 3. n - Number of points in the xa and ya arrays.
1735 C 4. yp1 - First derivative of the EOP function at the point i=1.
1736 C 5. ypn - First derivative of the EOP function at the point i=n.
1737 C
1738 C Output variables:

1751 C 2. p -
1752 C 3. qn -
1753 C 4. sig -
1754 C 5. un -
1755 C 6. u(NMAX) -

1757 C Programmer:
1758 C 93.11.22 Jim Ryan - Initial coding and modification for Calc.

1760 C Calc-like documentation inserted.
