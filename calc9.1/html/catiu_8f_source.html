166 C 3. ATIME
167 C
168 C 3.1 ATIME PROGRAM SPECIFICATION
169 C
170 C 3.1.1 ATIME IS THE UTILITY ROUTINE WHICH COMPUTES THE ATOMIC TIME
171 C FROM THE UTC TIME AND ALSO CALCULATES THE PARTIAL DERIVATIVE
172 C OF THE UTC TIME WITH RESPECT TO THE ATOMIC TIME.
173 C
174 C 3.2 ATIME PROGRAM INTERFACE
175 C
176  Real*8 UTC, XJD, AT, DUTCAT
177 C
178 C 3.2.1 CALLING SEQUENCE -
179 C INPUT VARIABLES:
180 C 1. UTC - THE UTC FRACTION OF THE UTC DAY. (SEC/SEC)
181 C 2. XJD - THE JULIAN DATE AT 0:00 UTC OF THE DATE IN QUESTION.
182 C OUTPUT VARIABLES:
183 C 1. AT - THE ATOMIC TIME FRACTION OF THE ATOMIC TIME DAY. (DAYS)
184 C 2. DUTCAT - THE PARTIAL DERIVATIVE OF THE UTC TIME WITH RESPECT TO
185 C THE ATOMIC TIME. (SEC/SEC)
186 C
187 C 3.2.2 COMMON BLOCKS USED -
188 C
189  Real*8 PI, TWOPI, HALFPI, CONVD, CONVDS, CONVHS, SECDAY
190  COMMON / cmath / pi, twopi, halfpi, convd, convds, convhs, secday
191  EXTERNAL cmathb
192 C VARIABLES 'FROM':
193 C 1. SECDAY - THE CONVERSION FACTOR OF COORDINATE TIME SECONDS PER
194 C COORDINATE TIME DAY. (SEC/DAY)
195 C
196  Real*8 ATMUTC(3), ROTEPH(2,20), A1UTC(3), A1DIFF(3)
197  COMMON / eopcm / atmutc, roteph, a1utc, a1diff
198 C VARIABLES 'TO/FROM':
199 C 1. ATMUTC(3) - THE 'TAI MINUS UTC' INFORMATION ARRAY. CONTAINS
200 C RESPECTIVELY THE EPOCH, VALUE, AND TIME RATE OF
201 C CHANGE OF 'TAI MINUS UTC'. (DAYS, SEC, SEC/SEC)
202 C
203  include 'ccon.i'
204 C VARIABLES 'FROM':
205 C 1. KATIC - THE ATIME UTILITY ROUTINE FLOW CONTROL FLAG.

209 C

223 C ATIME Program Structure
224 C
225 C Compute the atomic time fraction of the atomic time day.
226  at = + atmutc(2) / secday
227  1 + atmutc(3) * ( xjd - atmutc(1) )
228  2 + utc
229 C
230 C Compute the partial derivative of the UTC time with respect

232  dutcat = 1.d0 / ( 1.d0 + atmutc(3) )
233 C
234 C Check KATIC to see if the utility is to be turned off.
235  IF ( katic .NE. 1 ) GO TO 400
236  at = utc
237  dutcat = 1.d0
238 C

243 C

245  8 FORMAT(a,4d25.16/(7x,5d25.16))
246  WRITE(6,8)' SECDAY ',secday

248  9200 FORMAT (1x, "UTC = ", d30.16, /, 1x,
249  1 "XJD = ", d30.16, /, 1x,
250  2 "AT = ", d30.16, /, 1x,
251  3 "DUTCAT = ", d30.16 )
