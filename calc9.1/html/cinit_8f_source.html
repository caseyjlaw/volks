8 C 1.1.1 INITL obtains the mathematical and physical constants from the database
9 C and elsewhere and loads them into the common block 'cphys' for their
10 C use throughout the program. INITL also calls the input and
11 C initialization sections of the model modules and the necessary utility
12 C routines. Each section will obtain internally the model module parameter
13 C values from the database and initialize all such variables in the local
14 C common block. Each section will also put into the header a text message
15 C for each model module and necessary utility routine. INITL also
16 C initializes a counter which counts the observation number and writes a
17 C header text for the observation number, time tag, baseline
18 C identification, and source identification.
